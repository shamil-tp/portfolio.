Skylark

Premium Rent-A-Car Platform

Project Overview

Skylark is a full-stack, premium rent-a-car service web application that gives customers an easy, fast, and reliable way to book luxury vehicles online. Targeting the Indian market, it integrates Razorpay — India's leading payment gateway — for secure UPI, card, and net-banking payments.

The platform serves two audiences: customers who browse a luxury fleet (BMW, Benz), select rental dates, and pay securely online; and admins who manage the entire fleet, resolve customer issues, and analyse revenue through a dedicated portal.

How The Project Works

Skylark operates as a client-server architecture split across a Vue.js SPA frontend and a Node.js + Express REST API backend, with MongoDB as the data store and Razorpay handling the payment lifecycle.

1
User browses the fleet — Vue SPA calls Express REST API to fetch available cars, pricing, and availability windows.
2
User selects a car and fills the booking form (dates, duration) — frontend submits to API which creates a booking record in MongoDB with status pending.
3
Razorpay Order ID is generated server-side and returned to the frontend. The Vue app opens the Razorpay Checkout modal.
4
User completes payment via UPI / card / wallet. Razorpay sends payment signature back to the frontend.
5
Frontend passes signature to backend for cryptographic verification. On success, booking status is updated to confirmed in MongoDB.
6
An OTP Mailer workflow fires — confirmation email is sent to the user. Vue app displays the booking confirmation screen.
7
Admin portal reflects the new booking in real time — fleet availability updates, revenue analytics refresh via Chart.js dashboards.

Key Features

  • Online Car Browsing: Luxury fleet catalogue (BMW, Benz) with images, specs, daily pricing, and real-time availability.
  • Date-Based Booking System: Dynamic booking form with date picker, duration calculator, and conflict detection.
  • Razorpay Payment Integration: Secure payment via UPI, credit/debit cards, and net banking with server-side order creation and cryptographic verification.
  • JWT Authentication: Secure registration, login, OTP email verification, and role-based route access (user vs admin).
  • Admin Dashboard: Full fleet management, user overview, booking tracking, issue resolution portal, and revenue analytics with Chart.js.
  • Issue & Support System: Customers can raise issues; admins receive and resolve them through a dedicated ticket management interface.
  • OTP Mailer Workflow: Automated transactional emails for account verification, booking confirmations, and password resets.
  • Multi-Layout SPA: Separate AdminLayout, AuthLayout, and MainLayout for a clean, context-aware user experience.
  • Revenue Analytics: Visual revenue reports and fleet utilisation charts built with Chart.js in the admin panel.
  • Responsive Design: Tailwind CSS-powered UI works seamlessly across desktop, tablet, and mobile.
